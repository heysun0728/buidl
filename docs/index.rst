======
BUIDL
======

BUIDL enables rapid prototyping and development of Ethereum compatible DApps. It is a visual IDE for Solidity / Lity + HTML5 + JavaScript + web3 + ElasticSearch development. It supports

* Solidity smart contracts
* Lity rules engine and other extensions
* A gas free, fast finality, and high performance blockchain for deployment
* A dedicated blockchain smart contract data search engine
* Integrated web3 and data service API
* Other ppl can access your DApp directly from the web without wallet or cryptocurrency
